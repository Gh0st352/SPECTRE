POLY.PointWithinShape

Checks if point is within shape.

SPECTRE.POLY.PointWithinShape(P, Polygon) line 76

PointWithinShape. 

   1. Draw a horizontal line to the right of each point and extend it to infinity.
   2. Count the number of times the line intersects with polygon edges.
   3. A point is inside the polygon if either count of intersections is odd or point lies on an edge of polygon.
   4. If none of the conditions is true, then point lies outside.

Parameters:

  • P

    The vec2 to test whether inside polygon or not.

           P = {x=#,y=#}
  • Polygon

    Polygon to test P against. Table of Vec2 points. #Polygon must be >= 4

        Polygon = {
                [1] = {x = #, y = #},
                [...] = {x = #, y = #},
                [n] = {x = #, y = #},

Returns:

  1. true
  2. false
