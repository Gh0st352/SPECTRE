SPECTRE.WORLD.TargetMarker_CreateAtPointIfUnitsExistInZone(ZoneName, ObjectCategory, Table_UnitCategory, UnitCoalition, Coordinate, MarkCoalition, MarkerRadius, MarkText) line 108
Creates a Red Circle and Marker at the desired coord for specific team if specific units are detected in a zone

Parameters:

  • ZoneName Name of Zone to check if units exist within, EX. "Zone_Damascus"
  • ObjectCategory Object Category to scan for within zone, EX. Object.Category.UNIT
  • Table_UnitCategory Table of units types to scan for, EX. {Unit.Category.GROUND_UNIT}
  • UnitCoalition Coalition of units to scan for, EX. coalition.side.RED
  • Coordinate Coordinate to center Marker on
  • MarkCoalition Coalition to create the marker for, Coalition: All=-1, Neutral=0, Red=1, Blue=2. Default -1=All.
  • MarkerRadius Radium for the marker (meters)
  • MarkText Text to include in the marker

Returns:

    ReturnTable : {MarkerID,CircleID} - Table containing ID for created Marker and Circle
