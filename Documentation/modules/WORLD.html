SPECTRE.WORLD.TargetMarker_CreateAtPointIfUnitsExistInZone(ZoneName, ObjectCategory, Table_UnitCategory, UnitCoalition, Coordinate, MarkCoalition, MarkerRadius, MarkText) line 85
Create a red target marker at a specified coordinate if units of a given category and coalition exist within a zone.

Old function kept for sentimental value. Still works.

Parameters:

  • ZoneName string: The name of the zone to check for units.
  • ObjectCategory number: The category of the object (e.g., Object.Category.UNIT).
  • Table_UnitCategory table: A list of unit categories to scan for within the zone (e.g., {Unit.Category.GROUND_UNIT}).
  • UnitCoalition number: The coalition code of the units to check for (e.g., 0 = neutral, 1 = red, 2 = blue).
  • Coordinate OBJECT: The coordinate object where the marker should be created.
  • MarkCoalition number (optional): The coalition that should see the marker. Default is -1 (all coalitions).
  • MarkerRadius number (optional): The radius of the created circle around the marker. Default is 5000.
  • MarkText string (optional): The text to display on the marker. Default is an empty string.

Returns:

    table: A list containing the marker ID and circle ID.
