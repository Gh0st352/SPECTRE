SPECTRE.MENU.Buttons.Dispatcher

This function handles the dispatch of menu buttons depending on the provided type and player. It performs various checks including resource availability and player points balance before executing the dispatch. It also updates related resources, player points, and provides coalition-wide notifications.

Parameters:

  • MANAGER The central manager responsible for overseeing resources.
  • Player The player requesting the dispatch action.
  • Type The type of action being dispatched (e.g., "AIRDROP").
  • MarkerIndex The index of the marker associated with the dispatch.

Usage:

    SPECTRE.MENU.Buttons.Dispatcher(managerInstance, somePlayer, "AIRDROP", 1) -- Initiates an airdrop based on the specified marker for the provided player.
