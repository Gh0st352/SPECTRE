SPECTRE.HANDLERS:ParachuteCleanupInit() line 48
Initializes the Parachute Cleanup handler.

This function sets up an event handler to remove parachutes from the map upon landing after ejection. It listens to the EVENTS.LandingAfterEjection event and triggers _ParachuteCleanup function whenever such an event occurs.

Returns:

    HANDLERS.ParachuteCleanup self The updated ParachuteCleanup handler with event handling initialized.

Usage:

    SPECTRE.HANDLERS:ParachuteCleanupInit() -- Initializes the Parachute Cleanup event handling.
SPECTRE.HANDLERS:_ParachuteCleanup(ParachuteCleanupEvent) line 61
Handles the cleanup of parachutes on landing after ejection.

This function is triggered upon landing after ejection events. It logs the event details and, if the initiator (parachute) exists, attempts to destroy it. This function is designed to be used as an event handler and not called directly.

Parameters:

  • ParachuteCleanupEvent The event data associated with a landing after ejection.

Usage:

    Called internally as an event handler for EVENTS.LandingAfterEjection.
